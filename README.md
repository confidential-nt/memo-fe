# EasyLog Frontend

## 소개

구름 KDT 풀스택 개발자 1기 과정 우수 프로젝트 수상작입니다. 이 프로젝트는 기존 메모 및 일정 관리 어플리케이션의 장점은 가져가되 단점은 개선하고, 기존 앱에는 없는 편의 기능을 제공하는 웹 서비스 입니다.

## 기술 스택

- React
- Typescript
- React Query
- Tailwind CSS
- Vite
- Lodash
- Axios
- Dexie.js
- TinyMCE
- MUI
- 그 외 UI 관련 라이브러리들

## 설치 및 실행 방법

1. 저장소를 복제합니다:

   ```bash
   git clone https://github.com/confidential-nt/memo-fe.git

   ```

2. 프로젝트 폴더로 이동합니다:
   ```bash
   cd memo-fe
   ```
3. 필요한 의존성을 설치합니다:
   ```bash
   cd krampoline
   npm install
   ```
4. 개발 서버를 시작합니다:
   ```bash
   npm run dev
   ```

## 빌드 방법

1. 프로젝트를 빌드 합니다:
   ```bash
   npm run build
   ```
